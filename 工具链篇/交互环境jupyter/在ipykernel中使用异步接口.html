1. 在 ipykernel 中使用异步接口

在ipykernel 4.10+中默认运行,这就意味着在jupyter中使用协程并不需要显式的申明loop,只需要将协程使用ensure_future注册到loop上即可,使用的时候我们就需要关注下task的状态,当其状态是完成时我们就可以使用result接口获取到结果

import asyncio
import aiohttp

async def get_github():
    async with aiohttp.ClientSession() as session:
        response = await session.get('https://api.github.com')
        result = await response.json()
    return result

task = asyncio.ensure_future(get_github())

task.done()

True

result = task.result()

1.1. 直接使用await

在ipykernel 5.0+中新增了一组魔术命令用于控制cell处理异步代码的行为,这个魔术命令为%autoawait 如果不带参数,则可以查看当前autoawait的状态.

%autoawait

IPython autoawait is `on`, and set to use `asyncio`

如果带上参数,它就可以用于设置使用的lib

  • True/False表示是否使用这个特性
  • asyncio/curio/trio表示使用哪个包做为异步的时间循环
%autoawait False

%autoawait

IPython autoawait is `off`, and set to use `asyncio`

%autoawait True

%autoawait

IPython autoawait is `on`, and set to use `asyncio`
